Miscellaneous Plotting Functions¶

Module with functions for plotting IRT-related results.

catsim.plot.gen3D_dataset_scatter(title, items, path)[source]¶

Generate the item matrix tridimensional dataset scatter plot

Parameters:
  • title (string) – the scatter plot title, which will also be used as the file name
  • items (numpy.ndarray) – the item matrix
  • path (string) – the path to save the scatter plot
catsim.plot.plot_irt(a=1, b=0, c=0, title=None, ptype='icc', filepath=None)[source]¶

Plots ‘Item Response Theory’-related item plots

Parameters:
  • a (float) – item discrimination parameter
  • b (float) – item difficulty parameter
  • c (float) – item pseudo-guessing parameter
  • title (string) – plot title
  • ptype (string) – ‘icc’ for the item characteristic curve, ‘iif’ for the item information curve or ‘both’ for both curves in the same plot
  • filepath (string) – saves the plot in the given path
