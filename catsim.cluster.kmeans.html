K-means Clustering Algorithm¶

catsim.cluster.kmeans.kmeans(x, k, init_method='naive', iters=100, n_init=1, debug=False, metric='euclidean')[source]¶

Cluster a set of data points using the k-means algorithm.

Parameters:
  • x (numpy.ndarray) – a matrix in which columns are features and lines are observations
  • k (integer) – number of desired clusters
  • init_method (string) – the centroids initialization method. Can be one of the following: [‘naive’, ‘varCovar’, ‘ward’] in which ‘naive’ initializes the k centroids in a way that all features are within the minimum and maximum values of each feature of x; ‘varCovar’ [Eri11] selects two features that explain most of the dataset and iteratively selects the data points furthest from the center and form each other as the initial centroids; and ‘ward’ initializes the centroids using the means of k clusters generated via the hierarchical agglomerative clustering procedure that uses Ward function.
  • iters (integer) – number of maximum iterations if convergence is not reached before
  • n_init (integer) – number of re-initializations; the fuction returns the best of n_init results, chosen as the one with minimum sum of intra-cluster variances.
  • debug (boolean) – if using an interactive console, such as iPython, passing True allows for extra output messages separated by user input.
catsim.cluster.kmeans.naive_init(x, k)[source]¶

Initialize first \(k\) centroids by randomly choosing representatives from the \(x\) matrix

catsim.cluster.kmeans.var_covar_init(x, k)[source]¶

Centroid initialization method proposed by [Eri11].

Only two features of the dataset are used in this method. Feature \(X_1\) is the one that has the largest absolute coefficient of variation. \(X_2\) is the one that has the largest absolute correlation coefficient wrt \(X_1\).

Centroid \(c_1\) is selected as the farthest data point to the mean. \(c_i, i = 2, \ldots, k\) are selected as the farthest data points wrt \(c_{i-1}\)

[Eri11](1, 2) Erisoglu, M., Calis, N., & Sakallioglu, S. (2011). A new algorithm for initial cluster centers in k-means algorithm. Pattern Recognition Letters, 32(14), 1701–1705. http://doi.org/10.1016/j.patrec.2011.07.011
catsim.cluster.kmeans.ward_init(x, k)[source]¶

Ward initialization method for the centroids of k-means algorithm. In this method, the data is clustered hierarchically using Ward’s function, which is a greedy iterative method that joins the two clusters in a way that the resulting cluster has the minimum possible variance of all the other clusters that could possibly be merged in the previous step

After the \(k\) clusters are generated, their centroids are calculated and used to initialize k-means.
