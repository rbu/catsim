catsim.misc.plot.plot3D(points, clusters, title, centers=None)[source]¶

Plots 3D cluster charts

Parameters:
  • points (numpy.ndarray) – a matrix with the 3D locations of the data points
  • clusters (numpy.ndarray) – a list with the cluster memberships for each data point
  • title (string) – The title for the plot
  • centers (numpy.ndarray) – a matrix with the positions of the cluster centers, if they exist
